LettError EditThatNextMaster

Current release 1.5, April 2025

This tool switches quickly between UFOs and their various windows. Default keyboard settings to cmd-shift-[ and cmd-shift-].

Services

  • Glyph Window: maintains window size, position, layer, zoom and selection.
  • Font Window: maintains window size, position and glyph selection
  • Space Center: maintains window size, position, display attributes, the glyphs on show as well as the pre and post glyphs.
  • Single Window Mode

Shortcuts

  • Set the shortcuts in the RoboFont Preferences > ShortKeys > Extensions / EditsThatNextMaster

MetricsMachine

With the MetricsMachine extension installed, if the front window is a Metricsmachine window, EditThatNextMaster will select or open a MetricsMachine window for the next master. It will also copy the pair list and set the current pair.

Note

ETNM stops if the next UFO in line does not have the glyph we're looking at. We can't make a new (empty) glyph because that can propagate one glyph to UFOs that don't necessarily need it.
