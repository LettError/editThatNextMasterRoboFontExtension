LettError EditThatNextMaster

Current release 1.4, October 2021

This tool switches quickly between masters. Default keyboard settings to cmd-shift-[ and cmd-shift-].

Services

  • Glyph Window: maintains window size, position, layer, zoom and selection.
  • Font Window: maintains window size, position and selection
  • Space Center: maintains window size, position and the selected glyphs
  • Single Window Mode

Shortcuts

  • Set the shortcuts in the RoboFont Preferences > ShortKeys > Extensions / EditsThatNextMaster

Skateboard

With the Skateboard extension installed and a designspace file open, EditThatNextMaster will cycle between the available masters in the document.

MetricsMachine

With the MetricsMachine extension installed, if the front window is a Metricsmachine window, EditThatNextMaster will select or open a MetricsMachine window for the next master. It will also copy the pair list and set the current pair.
